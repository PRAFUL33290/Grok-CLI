Grok Build CLI

La nouvelle génération d'assistant de codage en terminal par xAI

Version 4.3 • Avril 2026 — Entièrement repensé

Grok n'est plus seulement un modèle. C'est maintenant une expérience complète en terminal ultra-puissante : TUI riche, agents parallèles, workflows réutilisables, intégrations externes, et bien plus.

Les grandes nouveautés

🧩

Skills — Workflows réutilisables

Des paquets de prompts intelligents qui encapsulent des procédures complètes. Plus besoin de répéter les instructions à chaque fois.

  • Créez des skills avec /skillify
  • Auto-capture depuis une session existante
  • Invocation automatique ou via /nom-du-skill
  • Partageables via plugins et Git
🤖

Subagents & Personas

Des agents enfants spécialisés qui travaillent en parallèle avec leur propre contexte et outils limités.

  • Personas : implementer, reviewer, researcher, security-auditor, test-writer...
  • Isolation via git worktrees
  • Chaînage intelligent avec resume_from
  • Limite de profondeur configurable
📋

Plan Mode

Phase de planification structurée obligatoire avant toute modification sur les tâches ambiguës.

  • Exploration libre du code
  • Écriture uniquement dans plan.md
  • Approbation utilisateur avant implémentation
  • Révision collaborative avec commentaires inline
🔌

MCP Servers (Model Context Protocol)

Intégrez n'importe quel service externe via le protocole standard : GitHub, Linear, bases de données, Sentry, etc.

  • Outils découverts dynamiquement avec search_tool
  • OAuth géré automatiquement
  • Configuration stdio ou HTTP/SSE
  • Scoped par projet ou global

Autres avancées majeures

🔌

Plugins

Paquets partageables contenant skills + agents + hooks + MCP. Installation via Git ou marketplace.

🖥️

Agent Mode (ACP)

Intégration native dans Zed, Neovim, Emacs via le protocole Agent Client Protocol. 72 méthodes d'extension.

⚙️

Background Tasks + Scheduler

Lancez des serveurs, builds, tests en arrière-plan. /loop et scheduler pour tâches récurrentes.

📡

Headless & CI/CD

Mode non-interactif avec sortie JSON ou streaming NDJSON. Parfait pour pipelines et scripts.

🧠

Mémoire cross-session

Recherchez dans l'historique de toutes vos sessions passées avec memory_search.

🎨

Theming & Vim mode

Interface entièrement personnalisable. Support complet Vim (j/k, folds, etc.) et configuration fine.

Commandes essentielles

grok Lance l'interface TUI interactive
grok -p "ta tâche" --yolo Mode headless (scripts / CI)
/skillify Capture automatiquement un workflow en skill
/plugins Gère les plugins, hooks et marketplace
grok mcp add ... Ajoute un serveur MCP (GitHub, Linear...)
grok agent stdio Mode ACP pour intégration IDE
/loop 5m "vérifie les tests" Tâche récurrente toutes les 5 minutes
grok inspect Liste skills, plugins, MCP, agents...

Raccourcis clavier puissants

Ctrl + L Plugins / MCP
Ctrl + T Panneau des tâches
Ctrl + G Mettre en arrière-plan
Ctrl + O Mode YOLO (auto-approve)
Ctrl + N Nouvelle session
Ctrl + Shift + E Toggle tous les thinking blocks
